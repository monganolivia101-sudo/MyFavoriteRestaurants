🍽️ My Favorite Restaurants

Mobile Web Application — Course Project Documentation

Student: Olivia Mongan  |  Mobile Application Development

1. Purpose and Description

My Favorite Restaurants is a mobile web application built using HTML5, CSS3, and the jQuery Mobile framework. The purpose of the app is to give users a fun, easy-to-use guide to a curated list of favorite local and national restaurants. Users can browse featured restaurants, search by name or cuisine type, save their own personal favorites using web storage, and leave reviews. The app is fully responsive and works on phones, tablets, and desktop browsers.

2. Website Structure / Site Map

The application is a single HTML file using jQuery Mobile's multi-page structure. All pages share a consistent header, navigation bar, and footer. Users can navigate between pages using the nav bar, the slide-in panel, or the back button.
